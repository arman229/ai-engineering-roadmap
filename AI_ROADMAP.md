 
# 6-Month AI Engineer Practical Roadmap (2 Hours / Day)

## Month 1 — Python for AI + Data Analysis

Goal: Become comfortable working with **data in Python**.

### Week 1 — Python Foundations

**Topics**

* Python basics
* Lists, dictionaries, tuples
* Functions
* File handling
* Virtual environments

**Daily Tasks**

* Day 1: Install Python, VSCode, Jupyter. Write simple scripts.
* Day 2: Practice loops and functions.
* Day 3: Work with lists and dictionaries.
* Day 4: Read and write CSV files.
* Day 5: Practice solving 10 Python problems.
* Day 6: Build a small script analyzing text files.
* Day 7: Review and practice.

**Mini Project**
Build a **Python script that analyzes a CSV dataset**.
